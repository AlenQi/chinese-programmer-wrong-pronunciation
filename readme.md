中国程序员容易发音错误的汉字

| 词语  | 正确发音 | 错误发音 | 读法含义 |
| ---- | ------- | ------- | ------ |
| 密钥  | ✅ [mì yào] | ❌ [mì yuè] | 密码学中：密钥（secret key）——秘密的钥匙；私钥（private key）——私有的钥匙；公钥（public key）——公开的钥匙。密钥就是秘密钥匙的简称，故读作【 mì yào】无误 |
